# 闲鱼 Web 版使用指南

## 关键页面

### 1. 首页
- **URL**: `https://www.goofish.com/`
- **功能**: 浏览推荐商品、搜索、查看分类
- **关键元素**:
  - 搜索框：输入关键词搜索商品
  - 分类导航：手机数码、服饰、家具等
  - 登录入口：未登录时显示

### 2. 消息中心
- **URL**: `https://www.goofish.com/im`
- **功能**: 查看所有聊天消息
- **页面结构**:
  ```
  ┌─────────────────────────────────────┐
  │ 左侧：联系人列表                      │
  │ - 显示最近消息预览                    │
  │ - 未读消息数 (99+)                    │
  │ - 交易状态标签                        │
  ├─────────────────────────────────────┤
  │ 右侧：聊天窗口                        │
  │ - 对话历史                            │
  │ - 订单信息卡片                        │
  │ - 输入框 + 发送按钮                    │
  └─────────────────────────────────────┘
  ```

### 3. 个人中心
- **URL**: `https://www.goofish.com/personal`
- **功能**: 管理个人信息和商品
- **主要板块**:
  - 个人资料：用户名、粉丝、关注、简介
  - 宝贝管理：我发布的、我卖出的
  - 交易记录：我买到的
  - 收藏：我的收藏

### 4. 订单页面
- **URL**: `https://www.goofish.com/bought`
- **功能**: 查看买入/卖出订单
- **订单分类**:
  - 全部 | 待付款 | 待发货 | 待收货 | 待评价 | 退款中

### 5. 发布闲置
- **URL**: `https://www.goofish.com/publish`
- **功能**: 发布新商品
- **填写要素**:
  | 字段 | 说明 | 必填 |
  |------|------|------|
  | 宝贝图片 | 上传商品图片 | ✅ |
  | 宝贝描述 | 商品详情，最多 1500 字 | ✅ |
  | 商品规格 | 品牌、成色等 | ❌ |
  | 价格 | 售价 | ✅ |
  | 原价 | 展示折扣 | ❌ |
  | 发货设置 | 包邮/按距离/一口价/无需邮寄 | - |
  | 宝贝所在地 | 选择城市 | - |

## 订单状态说明

| 状态 | 说明 | 应对操作 |
|------|------|---------|
| 待付款 | 客户已下单但未付款 | 等待或提醒付款 |
| 待发货 | 已付款，等待卖家发货 | 发送交付链接/快递发货 |
| 待收货 | 已发货，等待买家确认 | 引导确认收货 |
| 待评价 | 交易完成 | 邀请评价 |
| 交易成功 | 已完成 | 可邀请好评 |
| 交易关闭 | 退款/取消 | 查看钱款去向 |

## 聊天对话识别

### 从对话中提取关键信息

1. **订单信息**
   - 商品名称
   - 价格
   - 数量

2. **交付信息**
   - 交付方式：堂食/外卖/自提
   - 取餐码/兑换码
   - 兑换链接

3. **交易状态**
   - `已读` - 消息已读
   - 时间戳 - 最后活跃时间

## 常见元素定位（Snapshot UID）

> ⚠️ 注意：UID 在每次页面刷新后会变化，需重新获取快照

### 消息页面
| 元素 | 查找方式 |
|------|---------|
| 联系人列表 | `complementary` 区域内的 `StaticText` |
| 聊天输入框 | `textbox` 类型，`multiline` 属性 |
| 发送按钮 | `button` 类型为 `发 送` |
| 未读标记 | `image` 类型为 `reminder` |

### 订单页面
| 元素 | 查找方式 |
|------|---------|
| 订单状态 | `StaticText` 显示状态文字 |
| 联系卖家 | `button` 类型为 `联系卖家` |
| 确认收货 | `button` 类型为 `确认收货` |
| 去评价 | `button` 类型为 `去评价` |

## 注意事项

1. **反自动化机制**: 闲鱼有风控，避免高频操作
2. **敏感操作**: 发送链接、收款等需用户确认
3. **UID 变化**: 每次刷新页面 UID 重新生成
4. **登录态**: 保持登录状态，避免频繁登录
5. **客服态度**: 保持友好、专业、及时
